객체 실습 문제

문제 1

다음 입력 버튼을 클릭하면 이름, 나이, 주소를 prompt()로 입력받은 후 입력 정보를 member라는 객체에 저장한다.
이후 출력 버튼을 클릭하면 member 객체 안에 있는 정보를 출력 창에 출력되도록 만들어보자.
(단, 입력을 받지 않은 상태에서 출력 버튼을 클릭하면 '입력을 먼저 해주세요.'라는 알림을 띄우도록 한다.)

ex)
이름 : 홍길동
나이 : 20세
주소 : 서울시 테헤란로

출력창

(단, 학생 정보는 객체로 배열에 저장하여 테이블에 입력하세요.)

이름 나이 주소
