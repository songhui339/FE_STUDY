JavaScript 함수

1. 함수 선언

1) 선언적 함수

function 키워드에 함수명을 작성하여 선언하는 방법이다

2) 익명 함수

함수명을 작성하지 않고 선언하는 방법이다. (이벤트 핸들러 등록 시 주로 사용)

3) 화살표 함수 (Arrow Function)

ES6에 도입된 함수 선언 방법으로 function 키워드 대신에 => 를 사용해서
좀 더 간략한 방법으로 함수를 선언할 수 있다
