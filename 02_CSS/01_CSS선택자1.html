CSS 선택자 1

1. 전체 선택자(*)

HTML 문서 안의 모든 태그를 선택할 때 사용한다

2. 태그 선택자(태그명)

HTML 문서 안의 같은 태그를 모두 선택할 때 사용한다

3. 아이디 선택자(#아이디)

  1. 아이디 선택자 테스트 1
  2. 아이디 선택자 테스트 2
  3. 아이디 선택자 테스트 3
  4. 아이디 선택자 테스트 4
  5. 아이디 선택자 테스트 5

4. 클래스 선택자(.클래스)

HTML 문서 안에 해당하는 클래스 속성 값을 가진 태그들을 선택할 때 사용한다

  1. 클래스 선택자 테스트 1
  2. 클래스 선택자 테스트 2
  3. 클래스 선택자 테스트 3
  4. 클래스 선택자 테스트 4
  5. 클래스 선택자 테스트 5
